# MEMO NOTES

A simple web application for creating and managing memo notes.

## Features

* Create new notes
* Edit existing notes
* Work with multiple notes
* Save notes using Local Storage

## Technologies

* React
* JavaScript
* HTML
* CSS

## About

This project was created as part of a React course and focuses on building a simple notes application using React components and modern JavaScript.

## Run the Project

```bash
npm install
npm run dev
```
